# SupabaseLink
A Mathematica package for connecting to Supabase via the PostgREST REST API.

## Development Setup

To load the paclet directly from a local checkout (without installing it), run the following in Mathematica:

```mathematica
PacletDirectoryLoad["/path/to/SupabaseLink/"];
Get["SupabaseLink`"] // Quiet;
```

Replace `/path/to/SupabaseLink/` with the absolute path to the root of this repository on your machine.
